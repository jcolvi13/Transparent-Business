
Objective: The objective of this project is to build an agent that assesses companies on stock market performance and ethical metrics data to provide the user with recommendations for alternative investments that better align with their values without sacrificing financial investment. 

Current MVP Definition:  A user can input a company name or ticker along with their ethical preferences for investment, then our system returns an ethical analysis, financial summary, and alternative stock recommendation that align with both performance and values. 

What Has Been Built:
	The foundations of the agent have been built in VS code using codex. The fundamental inputs for the agent have also been created. 

Technical Approach: we built an AI agent that prompts free AI models using OpenRouter and aggregates data from specified data sets (HuggingFace and ESG)

Evidence of Progress: (Github with code and photos)

Current Limitations and Open Risks: Stock data set APIs cost money at scale. 

Plan for Phase 3: 
Part of our plan for phase 3 is to analyze when the agent fails or outputs inconsistent data. The to adjust prompt structure, input and outputs, skill files, and constraints to improve reliability of the agent.
